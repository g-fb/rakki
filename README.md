A very simple vertical manga reader. Supports folders and cbz, cbr, cbt, cb7, zip, rar, tar, 7z file formats.

### Settings
- Maximum image width
- Image spacing
- Upscale images to fit maximum width
- Hide scrollbar

### Actions
- Toggle fullscreen `f`
- Open file `1`
- Open folder `2`
- Scroll to the top `Ctrl+Home`
- Scroll to the bottom `Ctrl+End`
